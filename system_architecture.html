Multimodal AI Architecture
Neural Schema
The CardioGuardAI multimodal neural network fuses five distinct input streams through parallel encoder branches, merges them in a cross-attention fusion layer, and passes through constitutional AI safety rails before producing risk scores and empathetic guidance. AUC: 0.8883.
Input Layer
5 Multimodal Streams
Vitals dim 8 features
ECG length 2048 samples
Symptom tokens 512 max
History window 90 days
Demo embedding 32 dim
Encoder Layer
Parallel Branch Encoders
Vitals MLP 3-layer
ECG 1D-CNN + Transformer
Symptoms Clinical BERT
Temporal BiLSTM + Attn
Output per encoder 128-dim
Fusion Layer
Cross-Modal Attention
Input 5 × 128 = 640
Attention heads 8
Output dim 256
AUC achieved 0.8883
Regularization Dropout 0.3
Output Layer
5 Output Heads
Risk score 0.0 – 1.0
SHAP features Top-5 + direction
Aarogya text LLM conditioned
Escalation Binary threshold
Archive TimescaleDB
